# Employee Management System (SQL Project)

## 📌 Project Overview
This project is a SQL-based Employee Management System designed to manage employee records, job roles, salaries, and payroll operations efficiently. It also includes analytical queries for business insights.

## 🛠️ Technologies Used
- SQL (MySQL)
- DBMS Concepts

## 🗄️ Database Structure
The system consists of the following tables:

- JobDepartment – Stores job roles and department details
- SalaryBonus – Salary, bonus, and annual compensation
- Employee – Employee personal and job details
- Qualification – Employee qualifications and requirements
- Leaves – Employee leave records
- Payroll – Salary processing and reports

## 🔗 Relationships
- Employee linked to JobDepartment
- Salary linked to Job roles
- Payroll integrates Employee, Job, Salary, and Leaves

## 📊 Key Features
- Employee record management (CRUD)
- Salary and payroll tracking
- Leave management system
- Department-wise analysis
- Advanced SQL queries for insights

## 📈 SQL Analysis Included
- Total number of employees
- Department with highest employees
- Average salary per department
- Top 5 highest-paid employees
- Total salary expenditure
